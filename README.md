# aws-lambda-python

- virtualenv
```
pip install virtualenv --user
```
- cd aws-lambda-python
```
cd aws-lambda-python
```

- create env
```
python -m virtualenv .
```
- activate
```
source bin/activate
```
- install custom packages
```
pip install bs4 --user
```
- create lambda fuction
```
touch lib/python2.7/site-packages/lambda_function.py
```
- add code snippet
